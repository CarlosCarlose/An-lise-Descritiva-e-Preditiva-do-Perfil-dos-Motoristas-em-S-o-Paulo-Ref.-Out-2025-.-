# An-lise-Descritiva-e-Preditiva-do-Perfil-dos-Motoristas-em-S-o-Paulo-Ref.-Out-2025-.-
Análise descritiva e preditiva do perfil socioeconômico e geográfico de condutores categoria B com EAR em São Paulo, usando dados de emissão de CNH (out/2025), para identificar oportunidades no mercado automotivo e de locação.
